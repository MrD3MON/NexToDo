# Getting Started

## Prerequisites

- [Bun](https://bun.sh/) (or Node.js 18+)

## Installation

1. **Install dependencies**

```bash
bun install
```

2. **Setup Convex**

Run the setup script to create and connect a Convex project:

```bash
bun run dev:setup
```

3. **Environment variables**

Set Convex environment variables for auth and emails (via Convex Dashboard or CLI):

```bash
npx convex env set SITE_URL "http://localhost:3001"
npx convex env set GOOGLE_CLIENT_ID "your-google-client-id"
npx convex env set GOOGLE_CLIENT_SECRET "your-google-client-secret"
npx convex env set RESEND_API_KEY "re_your_resend_api_key"
npx convex env set FROM_EMAIL "noreply@yourdomain.com"
```

Copy other environment variables from `packages/backend/.env.local` to `apps/web/.env` if needed.

4. **Run the development server**

```bash
bun run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

5. **Create the First Admin**

To manage users and access admin features, you'll need an admin account:

- Create a normal account by signing in locally.
- Open the [Convex Dashboard](https://dashboard.convex.dev) for your project.
- Go to **Functions**, find `auth:createAdmin`, and run it with your `userId` (which you can find in the `users` table under **Data**).
- For subsequent users, admins can change roles directly within the app.

## Project Structure

```
NexToDo/
├── apps/
│   ├── web/           # React frontend (Vite + TanStack Router)
│   └── fumadocs/      # Documentation (this site)
├── packages/
│   ├── backend/       # Convex backend (boards, cards, auth, emails, etc.)
│   ├── config/        # Shared TypeScript config
│   └── env/           # Environment schema
└── task.md            # Development task tracker
```

## Available Scripts

| Script                | Description                         |
| --------------------- | ----------------------------------- |
| `bun run dev`         | Start web app and Convex dev server |
| `bun run build`       | Build all applications              |
| `bun run dev:web`     | Start only the web app              |
| `bun run dev:setup`   | Setup Convex project                |
| `bun run check-types` | Type-check all packages             |
